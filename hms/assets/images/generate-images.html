Instructions:

This page generates placeholder images for the HMS authentication system. To create actual images:

  1. Right-click on each image below
  2. Select "Save image as..."
  3. Save with the appropriate filename (patients.jpg, doctors.jpg, admin.jpg)
  4. Place the images in the hms/assets/images/ directory

Note: For production use, replace these with actual professional medical images.
